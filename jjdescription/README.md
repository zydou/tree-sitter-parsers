# tree-sitter-jjdescription

[Tree-sitter](https://tree-sitter.github.io/tree-sitter/) grammar for [`jj
desc`](https://docs.jj-vcs.dev/latest/cli-reference/#jj-describe) buffers.
Includes `diff` injection for `JJ: ignore-rest` comments and support for
[Conventional Commit](https://www.conventionalcommits.org/en/v1.0.0/) headers.
